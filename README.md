# Send-Mail-Using-SMTP

Simple Node application to send mail using smtp

## How to use

1. Clone this repo
```
git clone https://github.com/Lakshya-Saini/Send-Mail-Using-SMTP.git
```

2. Install dependencies
```
npm i
```

3. Create .env in root and add these variables
```
EMAIL_HOST=
EMAIL_AUTH_USER=
EMAIL_AUTH_PASS=
TO_EMAIL=
```

4. Run app
```
npm run dev
```
